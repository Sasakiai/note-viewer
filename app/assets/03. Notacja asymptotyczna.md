## O duże

`f(n) = O(g(n))`

Funkcja `f` rośnie co najwyżej tak szybko jak `g`.

To jest górne ograniczenie.

Przykład:

`n^2 + 5n + 2 = O(n^2)`

## o małe

`f(n) = o(g(n))`

Funkcja `f` rośnie istotnie wolniej niż `g`.

To jest ścisłe górne ograniczenie.

Przykład:

`n = o(n log n)`

## Θ

`f(n) = Θ(g(n))`

Funkcje mają ten sam rząd wzrostu.

Przykład:

`n^2 + 5n + 2 = Θ(n^2)`

## Ω duże

`f(n) = Ω(g(n))`

Funkcja `f` rośnie co najmniej tak szybko jak `g`.

To jest dolne ograniczenie.

Przykład:

`n^2 + 5n + 2 = Ω(n^2)`

## ω małe

`f(n) = ω(g(n))`

Funkcja `f` rośnie istotnie szybciej niż `g`.

To jest ścisłe dolne ograniczenie.

Przykład:

`n log n = ω(n)`

## Jak to czytać

- `O(g(n))` - nie szybciej niż `g`
- `o(g(n))` - wyraźnie wolniej niż `g`
- `Θ(g(n))` - tak samo szybko jak `g`
- `Ω(g(n))` - nie wolniej niż `g`
- `ω(g(n))` - wyraźnie szybciej niż `g`

## Krótkie przykłady

- `n^2 + 5n + 2 = O(n^2)`
- `n^2 + 5n + 2 = Θ(n^2)`
- `n^2 + 5n + 2 = Ω(n^2)`
- `n^2 + 5n + 2` nie jest `O(n)`, bo rośnie szybciej niż funkcja liniowa

## Hierarchia rzędów

Od wolniejszych do szybszych:

- O(1)
- O(log n)
- O(√n)
- O(n)
- O(n√n)
- O(n log n)
- O(n^k) dla stałego `k > 1`
- O(n^2)
- O(n^3)
- O(2^n)
- O(n!)
