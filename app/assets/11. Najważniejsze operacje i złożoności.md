## Lista jednokierunkowa

```
traverse     = O(n)
search       = O(n)
pushFront    = O(1)
popFront     = O(1)
insertAfter  = O(1), jeśli mamy wskaźnik
deleteAfter  = O(1), jeśli mamy wskaźnik
```

## Lista dwukierunkowa

```
traverse     = O(n)
search       = O(n)
pushFront    = O(1)
pushBack     = O(1), jeśli mamy tail
popFront     = O(1)
popBack      = O(1), jeśli mamy tail
deleteNode   = O(1), jeśli mamy wskaźnik
```

## Stos

```
push   = O(1)
pop    = O(1)
top    = O(1)
```

## Kolejka

```
enqueue   = O(1)
dequeue   = O(1)
front     = O(1)
```

## Deque

```
pushFront   = O(1)

```

## Kopiec binarny

```
findMin         = O(1)

```

## HashMap

Średnio:

```
search = O(1)
insert = O(1)
delete = O(1)
```

Pesymistycznie:

```
search = O(n)
insert = O(n)
delete = O(n)
```

## BST

```
search = O(h)
insert = O(h)
delete = O(h)
```

Dla dobrze zbalansowanego drzewa:

`O(log n)`

Pesymistycznie:

`O(n)`

## AVL

```
search    = O(log n)
insert    = O(log n)
delete    = O(log n)
rotacja   = O(1)
```

## Wyszukiwanie

### Wyszukiwanie sekwencyjne

```
W(n) = Θ(n)
A(n) = Θ(n)
S(n) = O(1)
```

### Wyszukiwanie skokami co k

```
W(n, k) = Θ(n/k + k)
A(n, k) = Θ(n/k + k)
S(n, k) = O(1)
```

Dla `k = √n`:

`Θ(√n)`

### Wyszukiwanie binarne

```
W(n) = Θ(log n)
A(n) = Θ(log n)
S(n) = O(1)
```

## Sortowanie

### SelectionSort

```
W(n) = Θ(n^2)
A(n) = Θ(n^2)
S(n) = O(1)
```

### InsertionSort

```
W(n) = Θ(n^2)
A(n) = Θ(n^2)
S(n) = O(1)
```

### Merge

```
W(n) = Θ(n)
A(n) = Θ(n)
S(n) = Θ(n)
```

### MergeSort

```
W(n) = Θ(n log n)
A(n) = Θ(n log n)
S(n) = Θ(n)
```

### QuickSort

```
W(n) = Θ(n^2)
A(n) = Θ(n log n)
S(n) = O(log n)
```

### CountSort

```
W(n, m) = Θ(n + m)
A(n, m) = Θ(n + m)
S(n, m) = Θ(n + m)
```
