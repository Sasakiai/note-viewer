#PJATK #ASD 

## Lista jednokierunkowa

```
traverse     = O(n)
search       = O(n)
pushFront    = O(1)
popFront     = O(1)
insertAfter  = O(1), jeśli mamy wskaźnik
deleteAfter  = O(1), jeśli mamy wskaźnik
```

## Lista dwukierunkowa

```
traverse     = O(n)
search       = O(n)
pushFront    = O(1)
pushBack     = O(1), jeśli mamy tail
popFront     = O(1)
popBack      = O(1), jeśli mamy tail
deleteNode   = O(1), jeśli mamy wskaźnik
```

## Stos

```
push   = O(1)
pop    = O(1)
top    = O(1)
```

## Kolejka

```
enqueue   = O(1)
dequeue   = O(1)
front     = O(1)
```

## Deque

```
pushFront   = O(1)
pushBack    = O(1)
popFront    = O(1)
popBack     = O(1)
```

## Kopiec binarny

```
findMin         = O(1)
insert          = O(log n)
delMin          = O(log n)
construct       = O(n log n)
fastConstruct   = O(n)
```

## HashMap

Średnio:

```
search = O(1)
insert = O(1)
delete = O(1)
```

Pesymistycznie:

```
search = O(n)
insert = O(n)
delete = O(n)
```

## BST

```
search = O(h)
insert = O(h)
delete = O(h)
```

Dla dobrze zbalansowanego drzewa:

`O(log n)`

Pesymistycznie:

`O(n)`

## AVL

```
search    = O(log n)
insert    = O(log n)
delete    = O(log n)
rotacja   = O(1)
```
